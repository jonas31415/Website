Hosting dieser Website

Für diese Website hab ich die Domain ohne Hosting bei Strato AG gekauft, da ich sie selber auf meinem RaspberryPi hosten will. Dazu installiert man zunächst mit dem Raspberry Pi Imager das Betriebssystem Raspberry Pi OS auf der entsprechenden SD-Karte. Anschließend kann man im Terminal des Raspberrys den Apache2 Webserver installieren. Anschließend ersetzt man im Verzeichnis /var/www/html die Index.html Datei mit den entsprechenden Dateien der Website. Um aus dem Internet darauf zugreifen zu können muss man nun noch Dynamische DNS einrichten, damit der Computer immer unter der selben Adresse erreichbar ist, auch wenn sich seinen IP-Adresse regelmäßig ändert.

Berechnen von Spacing-Ratios und Graphische Darstellung

In meiner Bachelorarbeit hab ich das Verhalten verschiedener Klassen von nicht-Hermiteschen Zufallsmatrizen untersucht. Dazu habe ich unter anderem die Eigenwerte berechnet und das Verhältnis aus nächst gelegenem Eigenwert zum übernächsten Eigenwert bestimmt. Diese Verhältnisse werden auch als Spacing-Ratio bezeichnet. Diese Verhältnisse kann man mit Hilfe einer gaußschen Kerndichteschätzung graphisch darstellen. Dazu wird die Matplotlib Bibliothek von Python genutzt. Hier befindet sich der entsprechende Code.

John Conway's Game of Life

Das Spiel des Lebens wurde vom Mathematiker John Conway entworfen. Es basiert auf zellulären Automaten, die hier durch die einzelnen Felder des Spielfeldes dargestellt werde. Jede Zelle kann genau zwei Zustände annehmen, nämlich tot oder lebendig. Die Spielregeln sind:

  • Eine tote Zelle mit genau drei lebenden Nachbarn wird in der nächsten Generation wiedergeboren.
  • Eine lebende Zelle mit weniger als drei lebenden Nachbarn stirbt in der Folgegeneration an Einsamkeit.
  • Eine lebende Zelle mit zwei oder drei lebenden Nachbarn bleibt weiterhin am Leben.
  • Eine lebende Zelle mit mehr als drei lebenden Nachbarn stirbt in der Folgegeneration an Überbevölkerung.
Der Code für das Spiel mit tkinter GUI (graphical user interface) ist auf GitHub zu finden.
